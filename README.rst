
`salt`
========

.. image:: ./icons/logo.png
    :width: 200px
    :height: 200px
    :align: center
    :alt: Salt

`salt` will solve your problem of where to start with documentation,
by providing a basic explanation of how to do it easily.

Look how easy it is to use:

    import project
    # Get your stuff done
    project.do_stuff()

Features
--------

- Be awesome
- Make things faster

Installation
------------

Install `salt` by running:

    install project
